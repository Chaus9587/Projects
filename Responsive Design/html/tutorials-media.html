@media query

Syntax

@media not/only mediatype and (mediafeature) and/not/or (mediafeature) ... {
    /* code */
}

Conditions ('mediafeature') in @media query

There are various conditions available to use in CSS @media query. For example, min-width/max-width, min-height/max-height, orientation and min-aspect-ratio/max-aspect-ratio are some of the commonly used conditions for the @media query.

Examples

Change the background color when the width of the browser is higher than 700px:

@media only screen and (min-width: 700px) {
    body {
         background-color: aqua;
    }
}

Change the backgound color when the width of the browser is larger than 700px while smaller than 1200px:

@media only screen and (min-width: 700px) and (max-width: 1200px) {
    body {
        background-color: rosybrown;
    }
}

You can mix things together, like using both and and or(,) logic operators:
(Change background color when the orientation of the browser is portrait/ width of the browser bewtween 550px and 750px)

@media only screen and (min-width: 550px) and (max-width: 750px), (orientation: portrait) {
    body {
         background-color: burlywood;
    }
}

You can also stack multiple @media query to create styling for different brwoser size/device:

@media only screen and (min-width: 300px) and (max-width: 450px) {
     body {
         background-color: antiquewhite;
     }
}
@media only screen and (min-width: 450px) and (max-width: 700px) {
     body {
         background-color: beige;
     }
}
@media only screen and (min-width: 700px) and (max-width: 1200px) {
     body {
         background-color: rosybrown;
     }
}
@media only screen and (min-width: 1200px) {
     body {
         background-color: darkgray;
     }
}

Of course you can do more than just background-color, the examples above is only for demonstrating how @media query works. You can also use the @media query to change the display, padding, margin etc.
