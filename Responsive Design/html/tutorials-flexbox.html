Flexbox

Introduction

CSS Flexbox is a web layout that has a high responsiveness, it is extremely functional when used with @media query.

How to set up a Flexbox container

You need to define a flex container with the flex items inside before you use it. The example below is for clear demonstration only and you can define your flex container in your own way.
(but display: flex in the container is required)

HTML:

<div class="flex-container">
    <div class="flex-item">1</div>
    <div class="flex-item">2</div>
    <div class="flex-item">3</div>
    <div class="flex-item">4</div>
</div>

CSS:

.flex-container {
    display: flex;
    height: 100vh;
}
.flex-item {
    padding: 10px 15px;
    margin: 0 5px;
    height: fit-content;
    background-color: rosybrown;
}

Output:

There you have it! A flex container is now created and we can start playing with it.

Aligning multiple flex items


(Note: the width/height: fit-content property are used to control the size of flex items and prevent them from accidental stretch through the whole width/height of the flex container.)


We will use the flex container and it's style properties that we just created above.

There are two main property to align the flex items inside the flex container, align-items and justify-content.



justify-content is used to align flex items horizontally.
Take a look at the examples and the respective output below.

CSS: (space-between)

.flex-container {
    justify-content: space-between;
}

Output:


CSS: (space-around)

.flex-container {
    justify-content: space-around;
}

Output:


CSS: (space-evenly)

.flex-container {
    justify-content: space-evenly;
}

Output:


CSS: (center)

.flex-container {
    justify-content: center;
}

Output:



align-items is used to align flex items vertically.

Consider the CSS code below is the current properties of the flex container and individual flex item.

.flex-container {
    display: flex;
    height: 100vh;
}
.flex-item {
    padding: 10px 15px;
    margin: 0 5px;
    height: fit-content;
    background-color: rosybrown;
}
.flex-item:nth-child(2) {
    padding: 15px;
}
.flex-item:nth-child(4) {
    padding: 20px 15px;
}

Here is the different examples of align-items:

CSS: (flex-start) (default)

.flex-container {
    align-items: flex-start;
}

Output:


CSS: (flex-end)

.flex-container {
    align-items: flex-end;
}

Output:


CSS: (baseline)

.flex-container {
    align-items: baseline;
}

Output:


CSS: (center)

.flex-container {
    align-items: center;
}

Output:


If you want to center a div, set both justify-content and align-items to center.

.flex-container {
    justify-content: center;
    align-items: center;
}

Output:


Aligning specific flex items

In some cases, developers may want to specify the location of a flex item. Which align-self can be used in this case. They will overwrite align-items.


Some examples of align-self. We will continue to use the CSS properties of the flex-contatiner and flex-items above.


CSS:

.flex-container {
    align-items: center;
}
.flex-item:nth-child(2) {
    align-self: flex-start;
}

Output:


CSS:

.flex-container {
    align-items: flex-start;
}
.flex-item:nth-child(2) {
    align-self: flex-end;
}

Output:

Controlling the width of flex items

You may have learnt the basic (min/max)-width property in CSS. However, in CSS flexbox, there are some other way to control the width of flex items in a flex container.


Consider the code below as the current styling of the flex itmms and the flex container.

HTML:

<div class="flex-container">
    <div class="flex-item">1</div>
    <div class="flex-item">2</div>
    <div class="flex-item">3</div>
    <div class="flex-item">4</div>
</div>

CSS:

.flex-container {
    display: flex;
}
.flex-item {
    padding: 10px 15px;
    margin: 0 5px;
    height: fit-content;
    background-color: cornflowerblue;
}

Output:


Similar to width, flex-basis set the initial length of flex items.

CSS:

.flex-item {
    flex-basis: 50px;
}

Output:


flex-grow/flex-shrink are used to set the width ratio of flex items. Note that the flex-shrink will only work when the total length of flex items are long enough to overflow the flex container itself.


To make all flex items stretch through the whole flex container in an equal length:

CSS:

.flex-item {
    flex-grow: 1;
}

Output:


More examples below.

CSS: (flex-grow)

.flex-item {
    flex-grow: 1;
}
.flex-item:nth-child(2) {
    flex-grow: 2;
}
.flex-item:nth-child(4) {
    flex-grow: 3;
}

Output:


CSS: (flex-shrink)

.flex-item {
    flex: 1 1 40%;
}
.flex-item:nth-child(2) {
    flex-shrink: 2;
}
.flex-item:nth-child(4) {
    flex-shrink: 3;
}

Output:


You may notice that I used the flex property. It is a shorthand for flex-grow, flex-shrink and flex-basis.
Syntax:

flex: |flex-grow| |flex-shrink| |flex-basis|;

Flex direction and wrapping of flex items

To increase the responsiveness of flex items, there are two flexbox properties that can achieve that. They are flex-direction and flex-wrap.

Consider the follow HTML and CSS code:

<div class="flex-container">
    <div class="flex-item">1</div>
    <div class="flex-item">2</div>
    <div class="flex-item">3</div>
    <div class="flex-item">4</div>
    <div class="flex-item">5</div>
    <div class="flex-item">6</div>
    <div class="flex-item">7</div>
    <div class="flex-item">8</div>
    <div class="flex-item">9</div>
</div>

CSS:

.flex-container {
    display: flex;
}
.flex-item {
    padding: 10px 15px;
    margin: 5px;
    width: fit-content;
    height: fit-content;
    background-color: palevioletred;
}

Let's see some examples of flex-direction.


CSS: (row) (default)

.flex-container {
    flex-direction: row;
}


CSS: (row-reverse)

.flex-container {
    flex-direction: row-reverse;
}


CSS: (column)

.flex-container {
    flex-direction: column;
}


CSS: (column-reverse)

.flex-container {
    flex-direction: column-reverse;
}


It is worth mentioning that when the property flex-direction: column/column-reverse is used, the effect of justify-content and align-items will be exchanged. (i.e justify-content will align flex items vertically, vice versa.).


flex-wrap determines the wrap direction when the flex container size is changing.


Adjust the pop-up window size if necessary to see the effect.

CSS: (nowrap) (default)

.flex-container {
    flex-direction: row;
    flex-wrap: nowrap;
}


CSS: (wrap)

.flex-container {
    flex-direction: row;
    flex-wrap: wrap;
}


CSS: (wrap-reverse)

.flex-container {
    flex-direction: row;
    flex-wrap: wrap-reverse;
}


There is also a shorthand property for flex-direction and flex-wrap, it is called flex-flow, syntax is shown below.

flex-flow: |flex-direction| |flex-wrap|;
